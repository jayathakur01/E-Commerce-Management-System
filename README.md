# E-Commerce Management System (Python)

## Overview

A console-based E-Commerce Management System developed in Python that simulates an online shopping platform with separate Admin and Member functionalities. The project demonstrates Object-Oriented Programming (OOP), data management using dictionaries, inventory tracking, and order processing.

## Features

### Admin Module

* Admin Login Authentication
* Add New Products
* View Product List
* Update Product Details
* Delete Products
* View Customer Orders

### Member Module

* User Registration and Login
* Browse Available Products
* Place Orders
* View Order History
* Update Profile Information

### Inventory Management

* Real-time stock tracking
* Automatic stock reduction after order placement
* Product availability management

## Technologies Used

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:

   ```bash
   cd E-Commerce-Management-System
   ```

3. Run the Python file:

   ```bash
   python main.py
   ```
